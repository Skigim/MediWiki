# MLTC Program Standards Chart

**MLTC Program Standards Chart 2025**  
*Income maximums, resource limits, SSI levels, and program standards*

---

## Overview

This comprehensive reference chart provides current financial and program standards for Medicaid and Long-Term Care (MLTC) programs in Nebraska, effective **January 1, 2025**.

**Key Information Included**:
- Income maximums for all MAGI and Non-MAGI programs
- Resource limits for various eligibility categories
- SSI Federal Benefit Rates (FBR)
- Spousal Impoverishment standards
- Facility standards and personal needs allowances
- Eligibility roles and application types

**Last Updated**: February 10, 2025

## Key Notes

**Federal Benefit Rate (FBR)**: When adding SSI income to Expert system, remember to also add SSI/FBR entry using this chart value. See [FAQ](FAQ.md) for detailed procedures.

**Income Limits**: All amounts are monthly maximums. Individuals with income at or below these limits may qualify for the respective programs (subject to other eligibility criteria).

**Resource Limits**: AABD/MA individual limit is $4,000; couple limit is $6,000. MSP programs have higher resource limits.

**Household Size**: For programs not listed beyond household size 10, contact policy staff for guidance.

**FPL Percentages**: Shown in table headers for reference; actual dollar amounts are what should be used for eligibility determination.

**Updates**: This chart is updated annually or when federal standards change. Always verify you are using the most current version.

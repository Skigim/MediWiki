# Medicaid Renewal Notice Language

**Nebraska Department of Health and Human Services | Medicaid & Long-Term Care**

## Overview

This document provides standardized language that can be used when a renewal notice isn't generated by NFOCUS. The template ensures that renewal notices include all required client rights and responsibilities information. Both English and Spanish versions are provided for use with approved Medicaid participants.

## Key Definitions

**Renewal Notice:** A notice informing Medicaid participants that their eligibility has been reviewed and renewed for continued coverage. The notice must include rights and responsibilities information.

**NFOCUS:** The Nebraska case management system used to process Medicaid applications and renewals. When NFOCUS generates a renewal notice automatically, no manual notice is needed.

**Notice Template:** A standardized NFOCUS notice format that includes client rights and responsibilities. Must be used instead of Speednote for renewal communications.

**Generic Notice:** An alternative NFOCUS notice format that includes client rights and responsibilities. Can be used when Notice Template is not appropriate.

**Speednote:** A quick communication tool in NFOCUS that does NOT include client rights and responsibilities. Should not be used for renewal notices.

**Electronic Data Sources:** Automated interfaces and databases used to verify information during the renewal process, such as income verification systems, SSA interfaces, and state databases.

---

## When to Use This Language

Use this standardized language when:
- A renewal notice isn't automatically generated by NFOCUS
- Manual notification of renewal approval is required
- Communicating renewal results to Medicaid participants

**Important:** Edit the template to include the names of all approved Medicaid participants and add comments if applicable.

---

## Notice Requirements

### Required Notice Format

✓ **Use Notice Template or Generic Notice** - These formats include client rights and responsibilities

✗ **Do NOT use Speednote** - Speednote doesn't include the client's rights and responsibilities

### Required Content

The renewal notice must include:
- Names of all approved Medicaid participants
- Their eligibility status
- Explanation of renewal process (electronic data sources, case record information, or new information provided)
- Requirement to inform the Department of any inaccurate information
- Instructions for reporting changes
- Signature line (optional for participant if all information is accurate)
- Manual reference: 477 NAC 3-007
- Link to ACCESSNebraska.ne.gov
- Comments section (if applicable)

## English Template

### Medicaid Renewal

**Notice Body:**

A renewal of eligibility for Nebraska Medicaid Programs has been completed. The following individual(s) remain eligible.

| Individual | Status |
|-----------|--------|
| John Doe | Eligible |

Renewals are completed using either electronic data sources, existing information in the case record or new information you provided during the renewal process. You are required to inform the Department if any of the information used to renew your eligibility is inaccurate. To do so, please sign and return this notice along with verification of any changes. If all information is current and accurate you are not required to sign and return this notice.

## Key Takeaways

✓ **Use Notice Template or Generic Notice format** - These include required client rights and responsibilities (do NOT use Speednote)

✓ **Edit template to include all approved Medicaid participants** - Replace "John Doe" with actual participant names and add rows as needed

✓ **Both English and Spanish versions available** - Use appropriate language based on participant preference

✓ **Signature is optional if information is accurate** - Participants only need to sign and return if reporting changes or inaccuracies

✓ **Always include manual reference 477 NAC 3-007** - Provide link to ACCESSNebraska.ne.gov for additional information

✓ **Participants must report inaccurate information** - If any renewal information is incorrect, they are required to inform the Department with verification

✓ **Renewals use multiple data sources** - Electronic data sources, existing case record information, or new information provided during renewal

✓ **Add comments section as needed** - Include additional information relevant to the specific renewal situation
